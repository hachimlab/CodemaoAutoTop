# CodemaoAutoTop

**もうそこに一番星 | 编程猫自动顶帖程序**

使用GitHub Action自动化完成

## 原理

在编程猫社区的论坛里，只要你发了一条评论，1999年的帖子都能到第一（实际上并没有1999年的帖子）

因此这个工具就是通过定时发评论+删评论，起到伪置顶的效果，让帖子一直在主页第一位

## 提示

顶帖必然会评论，也就是一天顶帖1000次就会有1000条同样的回复通知，请谨慎使用

## 部署

在GitHub上部署，请参考 [Wiki](https://github.com/hachimlab/CodemaoAutoTop/wiki)

在本地部署，请下载本仓库，装好对应依赖后设置定时任务即可
